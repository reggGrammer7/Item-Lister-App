# Item Lister

A lightweight item listing app built with vanilla JavaScript — no frameworks, no dependencies.

## Features

- Add items to a list via an input field
- Items display instantly on the page
- Built with plain HTML, CSS, and JavaScript

## What I Practiced

- DOM manipulation with vanilla JS
- Event listeners and form handling
- Dynamic rendering without a framework

## Run It

Just open `index.html` in your browser — no install needed.
